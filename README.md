# Superstore Sales Exploratory Data Analysis (EDA)

## Objective
Analyze sales performance, profitability, and customer behavior using the Superstore dataset to identify business insights and improvement opportunities.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Key Analysis
- Univariate, Bivariate, and Multivariate analysis
- Sales, Profit, Category, Region insights
- Business recommendations

## Dataset
Sample Superstore dataset (Kaggle)
